Get started

You need Node (and npm) and Yeoman for that.
1

Install generator-battle-backbone with npm

npm install -g generator-battle-backbone

You might need sudo for that

2

Scaffold your application using Yeoman

yo battle-backbone

All dependencies will be installed automatically using npm

3

Start a server

A small live reloading web server can be started using Grunt just by running 

grunt

4
